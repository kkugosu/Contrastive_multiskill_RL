# RL_META

## Installation
```
git clone https://github.com/kkugosu/RL_BASIC.git
```
## Experiment
```
python executable.py
```
you can choose belows
* environments 
* policy 
* batch size
* memory reset time
* train time per memory
* learning rate
* eligibility trace
* done penalty
* load previous model or not (not = 0 yes = 1)
